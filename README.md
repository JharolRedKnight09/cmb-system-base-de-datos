## 🌟 Descripción

**CMB System** es una plataforma administrativa diseñada para manejar y centralizar eficientemente la base de datos de la institución escolar. Su interfaz moderna permite gestionar la información clave de manera intuitiva y se actualiza en tiempo real, garantizando que el personal siempre tenga acceso a los datos más recientes.

## ✨ Características Principales

- 👨‍🎓 **Gestión Integral:** Administración de registros de forma organizada y eficiente (estudiantes, profesores, recursos).
- ⚡ **Sincronización en Tiempo Real:** Base de datos conectada en vivo, guardando cambios automáticamente.
- 🎨 **Interfaz de Usuario (UI) Moderna:** Sistema estético con diseño responsivo (adaptable a PC, tabletas y teléfonos).
- 🔒 **Seguridad y Privacidad:** Diseñado para manejar la información interna asegurando confidencialidad y control de acceso.

## 🛠️ Tecnologías Utilizadas

Esta aplicación se ha construido empleando herramientas de última generación en el desarrollo web frontend y backend (BaaS):

- **React 18** - Librería para la construcción de interfaces de usuario.
- **TypeScript** - Superconjunto de JavaScript que aporta tipado estático y reduce errores.
- **Vite** - Empaquetador extremadamente rápido (Build Tool).
- **Tailwind CSS** - Framework de utilidades CSS para dar estilos elegantes de manera veloz.
- **Firebase** - Backend as a Service (BaaS) utilizado para alojamiento de base de datos en tiempo real (Firestore) y/o despliegue.

## 🚀 Instalación y Uso Local

Para correr este proyecto en tu propia computadora, requerirás [Node.js](https://nodejs.org/) instalado. Luego sigue estos pasos:

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/JharolRedKnight09/cmb-system-base-de-datos.git
   ```
2. **Navega al directorio del proyecto**
   ```bash
   cd cmb-system-base-de-datos
   ```
3. **Instala las dependencias**
   ```bash
   npm install
   ```
4. **Levanta el servidor de desarrollo**
   ```bash
   npm run dev
   ```

## 🌐 Despliegue en la Nube
El proyecto está configurado y listo para ser desplegado mediante plataformas como **Vercel** o directamente en el **Hosting de Firebase**.
